synthit documentation
===================================================

synthit is an (alpha version) python package containing a variety of routines for MR image synthesis.
This package was developed by `Jacob Reinhold`_ and the other students and researchers of IACL_.

.. _IACL: http://iacl.ece.jhu.edu/index.php/Main_Page
.. _`Jacob Reinhold`: http://jcreinhold.github.io
